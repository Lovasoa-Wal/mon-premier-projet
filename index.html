Formation Git et GitHub

Bonjour dans mon apprentissage Git

Git est puissant et sécurisé. ceci est mon premier commit. commit c'est comme une photo du projet. A retenir, git ne supprime jamais une commit alors, on peut revenir à une ancienne version.

  1. modofier index.html
  2. lancer git diff (pour voir la difference)
  3. fais un commit avec message clair (ex: Amélioration du texte de la page d'accueil).

pour l'instant ce que vient de faire est de voir l'etat avec git status, préparer avec git add . et sauvegarder avec git commit -m "message"
